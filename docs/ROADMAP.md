# AerisLab Roadmap

## Version 0.2.0 (Q2 2026) - Stability & Performance

### Features
- [ ] Sparse matrix support for large systems (>100 bodies)
- [ ] Adaptive Baumgarte parameters
- [ ] Contact mechanics (sphere-plane, sphere-sphere)
- [ ] Wind field models (constant, turbulent)
- [ ] Multi-threading for force application

### Improvements
- [ ] Benchmark suite for performance tracking
- [ ] Memory profiling and optimization
- [ ] Better quaternion integration (mid-point, RK4)
- [ ] Warm-start for IVP solver

### Documentation
- [ ] Complete API reference (Sphinx)
- [ ] Physics theory document
- [ ] Tutorial videos
- [ ] Best practices guide

## Version 0.3.0 (Q3 2026) - Advanced Features

### Features
- [ ] Flexible bodies (modal reduction)
- [ ] Aerodynamic coefficient tables (interpolation)
- [ ] Atmospheric models (ISA, custom)
- [ ] Control systems integration (PID, state-space)
- [ ] Real-time visualization (Pygame/PyQt)

### Analysis Tools
- [ ] Sensitivity analysis utilities
- [ ] Monte Carlo simulation support
- [ ] Parameter optimization interface
- [ ] Frequency domain analysis

## Version 0.4.0 (Q4 2026) - Integration & Ecosystem

- [ ] ROS 2 integration

### Ecosystem
- [ ] Model library (common parachutes, payloads)
